# Simon-Ehrlich Wager

A visualization of the price data used to settle the [Simon-Ehrlich wager][wager].

## Building
```
$ yarn
$ yarn run dist
```

## Development Server
```
$ yarn start
$ open http://localhost:8000
```

## Clean up
```
$ yarn run clean
```
